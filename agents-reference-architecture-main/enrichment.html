Best Practices for Production-Grade Enrichment:
  • Web Unlocker: Use for high-scale, non-interactive scraping. Only pay for successful requests. Handles proxy management, headers, fingerprinting, and CAPTCHAs automatically.
  • Async Mode: For large batches or high concurrency, enable async requests to maximize throughput and reliability. Async mode is recommended for 1K+ parallel jobs and helps avoid 429 errors.
  • Browser API: If you need to interact with dynamic sites (e.g., fill forms, click, scroll), use Browser API for full browser automation at scale. Integrates with Puppeteer, Playwright, Selenium.
  • Troubleshooting: If direct API endpoint scraping fails, target the main webpage. For complex cases, use Browser API to intercept frontend API calls.
